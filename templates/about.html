Tech Stack Blueprint

Python Web Server

Flask framework manages route APIs, database transactions, session cookies, and Gemini connections.

Java Subprocess

ResumeParser.java binary utility validates PDF structure integrity and size metadata via subprocess.

SQL Database

SQLite database handles credentials, profile states, and application logs securely.

Gemini LLM API

Generative AI models match skills, evaluate ATS percentages, and recommended vacancies.

Why Choose This Platform?

This resume analyzer offers voice navigation control commands alongside native WebRTC face scans. By keeping code local and dependencies thin, it is optimized to run at high speed with 0% runtime errors.
